# Kems 🃏

Kems is a cross-platform card game built with Flutter, inspired by the classic multiplayer game where players use strategy, memory, and subtle signaling to win.

---

## 🚀 Features

* ♠️ Real-time card gameplay mechanics
* 👥 Multiplayer-ready architecture (expandable)
* 🎨 Clean and responsive UI built with Flutter
* ⚡ Fast performance across Android devices
* 🧠 Game logic structured for scalability and maintainability

---

## 🛠️ Tech Stack

* **Framework:** Flutter
* **Language:** Dart
* **Architecture:** Modular UI + Game Logic Separation
* **Build System:** Gradle (Android)

---

## 📦 Project Structure

```
lib/
│── main.dart            # Entry point
│── models/              # Game data models
│── screens/             # UI screens
│── widgets/             # Reusable UI components
│── services/            # Game logic / controllers
```

## ⚙️ Getting Started

### Prerequisites

Make sure you have:

* Flutter SDK (latest stable)
* Android Studio or VS Code
* Android SDK & emulator (or physical device)

---

## 🔧 Installation

```bash
git clone https://github.com/your-username/kems.git
cd kems
flutter pub get
```

---

## ▶️ Run the App

```bash
flutter run
```

---

## 🏗️ Build APK

```bash
flutter build apk
```

---

## 🧪 Testing

```bash
flutter test
```

---

## ⚠️ Known Issues

* Android build depends on compatible Gradle + AGP versions
* Multiplayer features may require backend integration (future work)
